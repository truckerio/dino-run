World Runner

World Runner is an interactive runner game made for an art project. The painting screen shows the game world, and visitors scan a QR code to play from their phone.

Created by Karanpreet Singh

Open game page View controls

Project Features

  • QR code starts the phone version of the game.
  • Painting mode runs an idle demo for the gallery screen.
  • Phone mode asks for a player name before the game starts.
  • Tap jumps, and swipe down ducks under flying jets.
  • High scores are saved so players can compete.
