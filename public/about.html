About the Project

World Runner connects a gallery display with a phone controller. The idea is to make the viewer part of the artwork instead of only watching it.

Art Mode vs Play Mode

Art Mode is the painting display. It shows the QR code and runs the idle demo so the screen still feels alive when nobody is playing.

Play Mode runs on the phone. It asks for the player's name, then the player controls the same runner with touch gestures.
